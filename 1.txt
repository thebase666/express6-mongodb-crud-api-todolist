 npm install eslint @eslint/js prettier eslint-config-prettier eslint-plugin-prettier -D
 复制文件 eslintrc.json 配置eslint   
 复制.prettierrc 配置prettier
 复制 .prettierignore 配置prettier忽略的文件

package.json中script中加代码
"lint": "eslint .",
"lint:fix": "eslint . --fix",
"format": "prettier --write .",
"format:check": "prettier --check .",
npm run lint检查  
npm run lint:fix 修复
npm run format：check 检查
npm run format 整理
